Counting Palm Tree from Satellite image¶

This is an attempt to count palm tree from satellite image. Few palm tree image will be use as samples, match it to a palm tree farm satellite image, then capture & count palm trees in the farm.

Filter high correlation result to get data point¶

Challenges:¶

With low pixel image download from Google search. Correlation thresholds set after trial and error test. Because of low pixel image, correlation thresholds need to be lower, enabling capture more data point. However, a low correlation threshold may cause wrong data point captured.

In [6]:
filter_result_1 = np.where(result_1>0.35)

Clean up duplicate data point¶

Data point are captured from multiple samples, same data point may captured multiple time, which can cause duplicate. Clean up is needed. When multiple data point captured is too near in range, we can ignore it. Only capture the first data point.

In [10]:
n = 0
m = 0
captured = ([],[])
for i in range(len(result[0])):
    this = result[0][i]
    if this not in range(m,n):
        captured[0].append(result[0][i])
        captured[1].append(result[1][i])
        m = this
        n = this + 4
captured =(np.asarray(captured[0]),np.asarray(captured[1]))
